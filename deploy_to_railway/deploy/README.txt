IGCSE Math Report Card Generator v2
Mr. Youssef Ahmed · Cambridge IGCSE

== SETUP (once only) ==
1. Install Python: https://www.python.org/downloads/
   Tick "Add Python to PATH"

== HOW TO USE ==
1. Double-click START_APP.bat
2. Browser opens automatically
3. Upload 3 CSVs → students load instantly
4. Filter by assistant name using the dropdown
5. Click any student to preview & edit their report
6. Add recommendations (or use quick templates)
7. Select students → Generate ZIP of PDFs
8. Use WhatsApp button to send directly to parent number

== WHAT'S AUTO-DETECTED ==
- All HW/Quiz/Mock assignments (no hardcoded limits)
- Student codes from Attendance sheet
- Parent phone numbers from Data sheet
- Assistant names for filtering

== GOOGLE SHEETS (future) ==

Attach the PDF manually before sending.
